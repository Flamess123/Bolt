# Bolt

This repository contains a React + TypeScript portfolio built with Vite and Tailwind CSS.

## Prerequisites
- Node.js 18+
- npm

## Setup
1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Linting and Build
Run ESLint:
```bash
npm run lint
```

Create a production build:
```bash
npm run build
```
